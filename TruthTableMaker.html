사용설명


1. 입력

  1. 변수 이름에는 '⊤', '⊥' 그리고 연산자를 제외한 어떤 글자도 사용할 수 있습니다.

  2. 변수의 수에는 제한이 없습니다.

    다만 변수 하나 당 2의 제곱으로 진리치의 경우의 수가 늘어나므로 컴퓨팅 성능의 한계를 고려해야합니다.

  3. 모든 입력버튼은 마지막 커서의 위치를 기준으로 작동합니다.

  4. 모든 입력버튼은 숫자, 특정 특수문자와 대응됩니다.

    대응되는 숫자, 특수문자를 입력하면 논리기호가 입력됩니다.

입력 숫자 입력 특수문자 입력 결과

2. 결과

  1. 변수 별 진리치 할당으로 해당하는 데이터만 하이라이트 하여 볼 수 있습니다.

  2. 테이블의 필드를 클릭하면 해당 열의 반전된 값을 볼 수 있습니다.

  3. PC버전에서는 테이블 뷰포트를 가로로 늘이거나 줄일 수 있습니다.

Manual


1. Input

  1. You can include any characters except '⊤' and'⊥' in the variable name.

  2. There is no limit to the number of variables.

    However,
    since the number of cases of truth value increases with the power of 2 per variable,
    the limit of computing power must be considered.

  3. All input buttons operate at the last cursor position.

  4. All input buttons correspond to numbers and specific special characters.

    Logical symbols are entered when you enter the corresponding number or special character.

2. Result

  1. By assigning truth values ​​to each variable, you can highlight specific rows.

  2. By clicking on a table field, you can see the inverted values ​​of that column.

  3. In the PC version, you can resize the table viewport horizontally.
